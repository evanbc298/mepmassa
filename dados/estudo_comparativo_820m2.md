## Alerta técnico importante (para conteúdo educativo)

Usar massa polimérica com **bloco comum (não retificado)** é um erro que anula toda a economia:

- A junta seca (1–3mm) não absorve variações de altura do bloco comum.
- Desvios milimétricos acumulam fiada por fiada → desaprumo de 15–20mm no topo da parede.
- Para corrigir, o reboco precisa chegar a 25–35mm → consome mais cimento do que o método tradicional.
- **Conclusão:** a economia do bloco barato é completamente eliminada no reboco corretivo.

O sistema correto é: **bloco retificado + argamassa polimérica + película fina de 3mm**.

## Aplicações validadas

- Alvenaria de vedação: **aprovado sem restrições**
- Alvenaria estrutural: uso restrito — NBR 16868 ainda não contempla juntas poliméricas milimétricas. Para edificações térreas, viável com sistemas híbridos (graute + armadura vertical de aço).
